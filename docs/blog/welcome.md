# Welcome to CyreneNameRoller Docs

Welcome to the official documentation site for **CyreneNameRoller**!

## What is CyreneNameRoller?

CyreneNameRoller is a powerful random name picker desktop application built with Vue 3 + Electron, featuring Windows 11 Fluent Design.

## Key Features

- 🎲 **Random Roller**: Multi-player mode, no-replacement, weighted balance algorithm
- 🃏 **Card Flip Mode**: 3D flip animation, quick multi-draw, persistent history
- 📊 **Statistics**: Real-time pick count, probability tracking
- 📋 **List Management**: CRUD, batch operations, import/export
- 🔒 **Data Security**: Password protection, dual-write persistence
- 🎨 **Personalization**: Dark mode, UI scale, font size, multi-language
