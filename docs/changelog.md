# 更新日志

## [26.0.0] - 2026-07-09

### 重构与性能
- 🚀 全新架构：基于 Vue 3 + Vite + Electron 全面重构，支持本地独立运行
- 🚀 数据双保险：Electron 文件系统 + localStorage 双写，意外崩溃数据也能自动恢复
- 🎨 Fluent Design (Win11) 设计语言，左侧 Dock 导航，操作更符合现代直觉

### 点名核心
- 🎲 随机点名：多人模式、禁止重复（不放回抽取）、平衡算法加权概率
- 🃏 翻牌点名：一键多抽、收牌历史持久化、切换名单不丢失状态
- 📐 Fritsch-Carlson 单调三次 Hermite 插值算法 + 可拖动曲线编辑器

### 数据安全
- 🔒 名单管理：CRUD、批量删除（10秒撤销）、导入/导出名单
- 🔒 密码保护关键数据操作，支持数据导出/导入/清除

### 统计与记录
- 📊 抽取记录：姓名、名单、来源、时间全程记录
- 📈 数据可视化：抽取次数、原始概率与平衡概率实时展示

### 界面与显示
- 🌗 深色模式、UI 缩放 (75%-200%)、名字字体大小调节
- 🌐 多语言：设置内切换中/英文

## 版本说明

### 版本号格式
`主版本号.次版本号.修订号`

- **主版本号**：重大架构变更或不兼容更新
- **次版本号**：新功能添加，向后兼容
- **修订号**：Bug 修复和小改进

### 版本代号
从 2.0.0 开始，每个版本都有一个代号：

- **Amphoreus**（安福洛斯）：希腊神话中的陶罐，象征包容与可能性
- 未来版本会继续使用神话或文学中的象征性名称
