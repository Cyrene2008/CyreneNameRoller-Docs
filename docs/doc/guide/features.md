# 功能指南

Cyreneの随机点名器是一款功能丰富的随机点名桌面应用，支持多种点名模式和数据管理功能。本文档将详细介绍所有功能。

## 随机点名功能

### 基本随机点名

- **单人模式**：每次抽取一名同学
- **多人模式**：每次抽取多名同学；允许重复时人数无上限，禁止重复时上限为当前候选人数
- **禁止重复**：本批不放回抽取，已抽中者从当前候选池移除
- **平衡算法**：以绝对次数极差 2 为固定软目标，每抽一人后更新状态，同时保证所有候选者概率大于零
- **小组模式**：按当前名单的小组进行单组或多组抽取

### 渐变名称动画

抽取结果以彩虹渐变色显示，色彩丰富生动。渐变颜色在 CSS 中定义为多种颜色的线性渐变，配合 32 秒循环位移动画，视觉效果流畅。

### 抽选结果强调动画

每次抽选完成后，结果名字会以从大到小的缩放动画（`scale(4)` → `scale(1)`）呈现，增强视觉冲击力。

### 名单切换

支持多个名单，可在随机点名界面实时切换不同名单进行抽取。

### 操作流程

1. 选择名单（可选）
2. 选择模式（单人/多人）
3. 如选择多人模式，配置抽取人数
4. 点击「开始」按钮
5. 查看抽取结果
6. 重复操作或切换名单

## 翻牌点名功能

### 3D 翻转卡牌

- **卡牌动画**：3D 翻转效果（`rotateY(180deg)`），视觉体验出色
- **一键多抽**：自动洗牌+翻牌，操作简便
- **自定义人数**：设置一次抽取的人数
- **收牌历史**：保存每次抽取的卡牌记录

### 操作流程

1. 设置抽取人数
2. 选择操作方式：
   - **手动模式**：点击「洗牌」发牌，逐张点击卡牌翻转
   - **一键多抽**：点击「快速抽取」按钮，自动完成洗牌+翻牌
3. 查看抽取结果
4. 查看收牌历史

### 卡牌管理

- **实时切换**：支持在抽取过程中切换名单
- **历史持久化**：收牌历史和已用名字自动保存，关闭应用后可恢复
- **重置功能**：清空当前牌面，重新开始

## 数据管理功能

### 数据统计

- **抽取次数**：统计每个名单的抽取次数
- **候选人数**：当前名单中的总人数
- **概率分布**：显示每位同学的被抽中概率
- **平衡概率**：启用平衡算法时，同时显示原始概率和平衡概率
- **排序**：按抽取次数降序排列

### 抽取记录

- **详细记录**：每次抽取的姓名、名单、来源、时间
- **来源标记**：区分「随机点名」和「翻牌点名」两种来源
- **时间格式化**：显示友好的时间格式
- **记录上限**：最多保存 500 条记录

### 名单管理

#### 基本操作
- **创建名单**：创建新的名单
- **重命名名单**：修改名单名称
- **删除名单**：删除不需要的名单

#### 人员管理
- **添加人员**：支持中文名和英文名
- **编辑人员**：修改人员的中文名和英文名
- **删除人员**：从名单中移除人员
- **批量删除**：支持全选后批量删除（10 秒撤销保护）

#### 白名单机制
- 系统内置白名单成员「再来一次」/「Again!」
- 白名单成员在批量删除时受保护，不会被删除
- 白名单不可由用户编辑

#### 导入导出
- **导出名单**：将单个名单导出为 `.json` 文件
- **导入名单**：从 `.json` 文件导入名单（添加为新名单）
- **无需密码验证**

### 小组管理

- 每份名单可建立独立小组，配置中英文组名、小组 ID 和成员
- 名单管理支持批量设置小组
- 点名页面可切换到小组模式，并与多人模式组合抽取

## 个性化设置

### 界面定制

- **UI 缩放**：调整界面大小（75% - 200%），适应不同屏幕
- **名字字体大小**：调整抽取结果中名字的显示大小（0.75x - 2.0x）
- **深色模式**：深色/浅色主题切换
- **语言切换**：中文 / English

### 名称颜色

- **渐变模式**（默认）：彩虹渐变色显示名字
- **自定义单色**：自定义浅色模式和深色模式下的名字颜色

### 性能优化

- **亚克力模糊**：开启/关闭背景毛玻璃效果
- **阴影效果**：开启/关闭元素阴影
- **过渡动画**：开启/关闭页面切换动画

### 数据安全

- **启用数据统计**：控制是否记录抽取统计数据；公平算法开启时自动启用并锁定
- **数据操作密码**：设置密码保护数据导出、导入、清空等敏感操作
- **密码加密**：使用 SHA-256 哈希存储，不可逆向还原

## 技术特性

### 多端支持

- **Tauri 版本**：轻量级，基于系统 WebView，安装包小
- **Electron 版本**：兼容性好，内置 Chromium
- **线上版本**：浏览器直接使用，无需安装

### 设计语言

- **Fluent Design**：采用 Windows 11 设计语言，界面简洁现代
- **自定义组件**：FluentButton、FluentCard、FluentModal 等自定义 UI 组件
- **响应式布局**：支持不同屏幕尺寸

### 数据持久化

- **Tauri**：通过 Rust 后端存储为 JSON 文件
- **Electron**：使用 electron-store
- **浏览器**：使用 localStorage
- **数据导出**：支持全量数据导出为 `.cyrene` 文件

### 桌面自动更新

- Electron 与 Tauri 均从设置页检查 GitHub Releases
- 安装包由原生层下载到系统临时目录，不经过浏览器下载页面
- 下载后校验官方地址、预期大小、最小体积和 EXE 文件头，通过后自动启动安装程序
