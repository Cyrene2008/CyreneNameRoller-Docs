# 功能指南

Cyreneの随机点名器是一款功能丰富的随机点名桌面应用，支持多种点名模式和数据管理功能。本文档将详细介绍所有功能。

## 随机点名功能

### 基本随机点名

- **单人模式**：每次抽取一名学生
- **多人模式**：每次抽取多名学生
- **禁止重复**：不放回抽取，确保同一名单中每人只被抽中一次
- **平衡算法**：被抽中次数越少，下次被抽中的概率越高，确保公平性

### 高级特性

- **彩虹名称动画**：抽取结果以彩虹渐变色显示，视觉效果生动
- **English Mode**：支持英文界面显示
- **名单切换**：支持多个名单，可实时切换
- **历史记录**：保存抽取历史，方便回顾

### 操作流程

1. 选择名单（可选）
2. 选择模式（单人/多人）
3. 点击 "开始" 按钮
4. 查看抽取结果
5. 重复操作或切换名单

## 翻牌点名功能

### 3D 翻转卡牌

- **卡牌动画**：3D 翻转效果，视觉体验出色
- **一键多抽**：自动洗牌+翻牌，操作简便
- **自定义人数**：设置一次抽取的人数
- **收牌历史**：保存每次抽取的卡牌记录

### 操作流程

1. 设置抽取人数
2. 点击 "一键多抽" 按钮
3. 等待洗牌动画完成
4. 逐个点击卡牌查看结果
5. 查看收牌历史

### 卡牌管理

- **实时切换**：支持在抽取过程中切换名单
- **历史持久化**：收牌历史保存在本地存储中
- **批量操作**：支持批量收牌和重置

## 数据管理功能

### 数据统计

- **抽取次数**：统计每个名单的抽取次数
- **概率分布**：显示每个学生的被抽中概率
- **平衡概率**：显示考虑平衡算法后的实际概率
- **图表展示**：可视化展示统计数据

### 抽取记录

- **详细记录**：每次抽取的姓名、名单、来源、时间
- **筛选功能**：按名单、时间、姓名筛选记录
- **搜索功能**：快速查找特定记录
- **导出功能**：将记录导出为 CSV 或 JSON 格式

### 名单管理

#### 基本操作
- **添加名单**：创建新的名单
- **编辑名单**：修改名单名称和描述
- **删除名单**：删除不需要的名单
- **批量删除**：支持多选批量删除（10秒撤销保护）

#### 学生管理
- **添加学生**：向名单中添加学生姓名
- **编辑学生**：修改学生姓名
- **删除学生**：从名单中移除学生
- **批量导入**：从文本文件批量导入学生姓名
- **导出名单**：将名单导出为文本文件

#### 数据安全
- **SHA-256 加密**：使用 SHA-256 算法保护敏感数据
- **本地存储**：所有数据保存在本地，保护隐私
- **自动备份**：支持自动备份功能

## 个性化设置

### 界面定制

- **UI 缩放**：调整界面大小（75%-200%），适应不同屏幕
- **名字字体大小**：调整抽取结果中名字的显示大小
- **主题切换**：深色模式 / 浅色模式，适应不同环境
- **语言切换**：中文 / English，支持多语言

### 功能配置

- **平衡算法开关**：启用/禁用平衡算法
- **English Mode**：切换英文显示模式
- **动画效果**：启用/禁用彩虹名称动画
- **音效设置**：启用/禁用抽取音效

### 快捷键

- **空格键**：快速开始抽取
- **Escape**：取消当前操作
- **Ctrl+S**：保存当前名单
- **Ctrl+Z**：撤销上一步操作
