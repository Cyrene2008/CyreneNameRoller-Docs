# 界面介绍

Cyreneの随机点名器采用 Windows 11 Fluent Design 设计语言，界面简洁现代，操作直观。本文档将详细介绍软件的各个界面部分。

## 主界面布局

软件主界面采用左侧导航栏设计，包含以下主要部分：

### 1. 左侧导航栏

导航栏位于窗口左侧，包含以下功能页面入口：

- **随机点名**：核心的随机抽取功能
- **翻牌点名**：3D 卡牌翻转动画点名
- **数据统计**：查看抽取次数和概率分布
- **抽取记录**：查看历史抽取记录
- **名单管理**：管理名单和人员
- **设置**：个性化配置选项
- **关于**：应用信息

导航栏支持收起/展开切换，收起后仅显示图标。

### 2. 自定义标题栏

应用使用自定义标题栏（无系统标题栏），包含：

- 应用名称显示
- 窗口控制按钮：最小化、最大化/还原、关闭

### 3. 主内容区

根据选择的页面显示不同的功能界面。页面切换带有滑动过渡动画（可在设置中关闭）。

## 功能页面详解

### 随机点名页面

核心的随机抽取功能：

- **模式切换**：单人模式 / 多人模式
- **人数调节**：多人模式下可设置抽取人数（2 人至名单人数上限）
- **禁止重复**：开关控制是否允许同一人被重复抽中
- **英文模式**：切换显示中文名/英文名
- **名单选择**：下拉菜单选择不同的名单
- **开始按钮**：启动随机抽取
- **结果展示区**：显示抽取结果，支持渐变名称动画和缩放强调动画

### 翻牌点名页面

具有 3D 翻转动画的卡牌点名：

- **英文模式**：切换显示中文名/英文名
- **名单选择**：下拉菜单选择不同的名单
- **洗牌按钮**：发牌，将卡牌面朝下放置
- **快速抽取**：一键完成洗牌+翻牌
- **重置按钮**：清空牌面，重新开始
- **卡牌区域**：显示卡牌，点击翻转查看结果
- **收牌历史**：底部区域显示已翻开的卡牌记录

### 数据统计页面

数据统计和可视化展示：

- **总计抽取次数**：所有名单的总抽取次数
- **候选人数**：当前名单中的总人数
- **统计列表**：
  - 每位同学的姓名
  - 被抽中次数
  - 原始概率
  - 平衡概率（启用平衡算法时显示）
  - 按抽取次数降序排列

### 抽取记录页面

查看历史抽取记录：

- **记录列表**：显示所有抽取记录
- **记录字段**：
  - 时间：抽取时间（友好格式显示）
  - 姓名：被抽中的同学姓名
  - 名单：所属名单名称
  - 来源：「随机点名」或「翻牌点名」
- **记录上限**：最多保存 500 条记录

### 名单管理页面

名单和人员的管理功能：

- **名单切换**：左侧列表显示所有名单，点击切换
- **添加名单**：创建新的名单
- **名单操作**：重命名、删除
- **人员列表**：显示当前名单中的所有人员
- **添加人员**：输入中文名和英文名
- **编辑人员**：修改人员信息
- **删除人员**：单个删除或批量删除
- **批量操作**：全选、批量删除（10 秒撤销保护）
- **导入导出**：导入/导出名单为 `.json` 文件
- **白名单标记**：白名单成员显示特殊标记，不可删除

### 设置页面

个性化配置选项，分为以下区域：

- **基本设置**：语言切换、深色模式
- **主题与显示**：名称颜色模式、界面缩放、名字字体大小
- **性能设置**：亚克力模糊、阴影效果、过渡动画
- **数据管理**：启用数据统计、数据操作密码、数据导出/导入/清空
- **平衡算法**：启用开关、曲线编辑器
- **更新日志**：版本更新历史

### 关于页面

显示应用的基本信息：

- 应用名称和版本号
- 应用描述
- 作者信息（Cyrene2008）
- GitHub 仓库链接
- 许可证信息（GPL v3.0）
- 版权声明
- 萌ICP备案号

## UI 组件

应用使用自定义的 Fluent Design 组件：

- **FluentButton**：按钮组件，支持 primary/secondary/subtle/danger 变体，sm/md/lg 尺寸
- **FluentCard**：毛玻璃效果卡片容器
- **FluentIcon**：基于 @iconify/vue 的 Fluent 图标
- **FluentInput**：输入框，支持前后缀插槽
- **FluentModal**：模态弹窗，支持 persistent 模式
- **FluentSelect**：下拉选择器
- **FluentToast**：右下角通知，支持操作按钮
- **FluentToggle**：开关切换
- **BalanceEditor**：平衡曲线编辑器，Canvas 绘制，可拖动控制点

## 界面操作技巧

### 随机点名操作

1. **快速点名**：在随机点名界面直接点击「开始」按钮
2. **多人模式**：切换到多人模式，设置抽取人数
3. **切换名单**：在名单选择下拉菜单中选择不同名单
4. **禁止重复**：开启后确保每人只被抽中一次

### 翻牌点名操作

1. **手动翻牌**：点击「洗牌」发牌，逐张点击卡牌翻转
2. **快速抽取**：点击「快速抽取」按钮，一键完成
3. **切换名单**：在名单选择下拉菜单中选择不同名单
4. **重置牌面**：点击「重置」按钮清空牌面

### 数据管理操作

1. **查看统计**：在数据统计页面查看抽取次数和概率
2. **查看记录**：在抽取记录页面查看历史记录
3. **管理名单**：在名单管理页面进行名单的增删改查
4. **导入导出**：使用导入/导出功能备份名单数据
