# 界面介绍

Cyreneの随机点名器采用 Windows 11 Fluent Design 设计语言，界面简洁现代，操作直观。本文档将详细介绍软件的各个界面部分。

## 主界面布局

软件主界面采用侧边导航栏设计，包含以下主要部分：

### 1. 侧边导航栏

侧边导航栏位于窗口左侧，包含以下功能选项卡：

- **随机点名**：主要的随机抽取功能
- **翻牌点名**：3D 卡牌翻转动画点名
- **数据管理**：查看统计数据、抽取记录和名单管理
- **设置**：个性化配置选项

### 2. 顶部标题栏

显示软件名称和窗口控制按钮（最小化、最大化/还原、关闭）。

### 3. 主内容区

根据选择的选项卡显示不同的功能界面。

## 功能界面详解

### 随机点名界面

核心的随机抽取功能：

- **模式选择**：单人模式 / 多人模式
- **开始按钮**：启动随机抽取
- **结果展示区**：显示抽取结果，支持彩虹名称动画
- **历史记录**：显示最近的抽取记录
- **名单选择**：选择不同的名单进行抽取

### 翻牌点名界面

具有 3D 翻转动画的卡牌点名：

- **抽取人数**：设置一次抽取的人数
- **一键多抽**：自动洗牌+翻牌
- **卡牌展示区**：显示卡牌，点击翻转查看结果
- **收牌历史**：查看历史抽取记录
- **名单切换**：实时切换名单

### 数据管理界面

数据统计和管理功能：

- **统计面板**：
  - 抽取次数统计
  - 概率分布
  - 平衡概率显示
- **抽取记录**：
  - 姓名、名单、来源、时间
  - 支持筛选和搜索
- **名单管理**：
  - 名单的增删改查 (CRUD)
  - 批量删除（10秒撤销保护）
  - 导入/导出功能
- **数据安全**：SHA-256 密码保护

### 设置界面

个性化配置选项：

- **UI 缩放**：调整界面大小（75%-200%）
- **字体设置**：调整名字字体大小
- **主题切换**：深色模式 / 浅色模式
- **语言切换**：中文 / English
- **平衡算法**：启用/禁用被抽中次数越少概率越高的算法
- **English Mode**：切换英文显示模式

## 界面操作技巧

### 随机点名操作

1. **快速点名**：在随机点名界面直接点击 "开始" 按钮
2. **多人模式**：切换到多人模式，设置抽取人数
3. **查看历史**：点击历史记录查看之前的抽取结果
4. **切换名单**：在名单选择下拉菜单中选择不同名单

### 翻牌点名操作

1. **设置人数**：在翻牌点名界面设置抽取人数
2. **一键抽取**：点击 "一键多抽" 按钮进行批量抽取
3. **翻转卡牌**：点击卡牌查看抽取结果
4. **查看历史**：点击 "收牌历史" 查看历史记录

### 数据管理操作

1. **查看统计**：在数据管理界面查看抽取次数和概率
2. **筛选记录**：使用筛选功能查找特定记录
3. **管理名单**：点击 "名单管理" 进行名单的增删改查
4. **导入导出**：使用导入/导出功能备份名单数据

### 设置操作

1. **调整缩放**：拖动 UI 缩放滑块调整界面大小
2. **切换主题**：点击深色/浅色模式切换按钮
3. **更改语言**：在语言下拉菜单中选择语言
4. **平衡算法**：勾选/取消勾选平衡算法选项

## 故障排除

### 界面显示问题

如果遇到界面显示问题，可尝试以下解决方法：

1. **调整缩放**：在设置中调整 UI 缩放比例
2. **重启应用**：关闭并重新启动应用程序
3. **更新显卡驱动**：确保显卡驱动是最新的

### 数据丢失问题

如果遇到数据丢失，请尝试以下方法：

1. **检查备份**：查看是否有导出的名单备份
2. **查看记录**：在抽取记录中查找历史数据
3. **联系支持**：如无法解决，请访问 GitHub Issues 寻求帮助
