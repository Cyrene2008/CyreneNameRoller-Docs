# 基本设置

本节介绍 Cyreneの随机点名器的所有可配置选项。所有设置均在「设置」页面中进行修改，修改后即时生效。

## 基本设置

### 语言切换

切换界面显示语言：

- **中文**：默认语言
- **English**：英文界面

切换方式：点击语言开关即可即时切换，无需重启应用。影响范围包括菜单、按钮、提示信息等界面文本。

### 深色模式

切换深色/浅色主题：

- **浅色模式**：适合白天或明亮环境使用
- **深色模式**：适合夜晚或暗光环境使用

切换方式：点击深色模式开关即可即时切换。

## 主题与显示

### 名称颜色模式

控制点名结果中名字的显示颜色：

- **渐变模式**（默认）：名字以彩虹渐变色显示，色彩丰富生动
- **自定义单色**：使用自定义颜色显示名字

选择「自定义单色」后，可分别设置浅色模式和深色模式下的名字颜色：

- **浅色模式颜色**：通过颜色选择器自定义
- **深色模式颜色**：通过颜色选择器自定义

### 界面缩放

调整界面整体大小，适应不同屏幕分辨率：

- **可选值**：75%、100%（默认）、125%、150%、175%、200%
- **实现方式**：通过 CSS 缩放实现，界面元素等比放大缩小

### 名字字体大小

调整点名结果中名字的显示大小：

- **可选值**：0.75x、1.0x（默认）、1.25x、1.5x、1.75x、2.0x
- **仅影响**：点名结果区域的名字文字大小，不影响其他界面元素

## 性能设置

以下设置用于在低性能设备上优化运行流畅度：

### 亚克力模糊

控制界面背景的毛玻璃模糊效果：

- **开启**：界面背景呈现 Windows 11 风格的亚克力模糊效果
- **关闭**：关闭模糊效果，提升低性能设备的运行流畅度

### 阴影效果

控制卡片和按钮的阴影：

- **开启**：界面元素显示阴影效果，增强层次感
- **关闭**：关闭阴影效果，减少渲染负担

### 过渡动画

控制页面切换和元素过渡动画：

- **开启**：页面切换时显示滑动过渡动画
- **关闭**：关闭过渡动画，页面直接切换，提升响应速度

## 数据管理

### 启用数据统计

控制是否记录抽取统计数据：

- **开启**（默认）：自动记录每次抽取的次数和概率分布
- **关闭**：停止记录统计数据，已有的统计数据保留但不再更新

### 数据操作密码

设置密码以保护敏感数据操作：

- **保护范围**：数据导出、数据导入、清空记录、清除全部数据
- **加密方式**：密码以 SHA-256 哈希存储，不可逆向还原
- **三种模式**：
  - **设置密码**：首次设置密码
  - **修改密码**：需先验证旧密码，再设置新密码
  - **验证密码**：执行受保护操作时需要验证

### 数据导出

将所有数据导出为 `.cyrene` 文件（JSON 格式）：

- **包含内容**：设置、名单、统计数据、抽取记录、平衡算法配置、密码哈希
- **需要密码验证**

### 数据导入

从 `.cyrene` 文件导入数据：

- **注意**：导入会覆盖当前所有数据
- **需要密码验证**
- **导入后需手动重启应用**

### 清空记录

清空所有抽取记录：

- **需要密码验证**
- **不影响**名单和统计数据

### 清除全部

清除所有数据并恢复默认设置：

- **需要密码验证**
- **不可恢复**，操作前请确认

## 平衡算法

### 启用平衡算法

控制随机抽取的公平性：

- **开启**：使用平衡算法，被抽中次数越少的同学概率越高
- **关闭**：完全随机抽取，每次概率相同

### 曲线编辑器

自定义平衡算法的概率调整曲线：

- **可视化编辑**：在 Canvas 上拖动 3 个控制点调整曲线形状
- **算法原理**：基于 Fritsch-Carlson 单调 Hermite 三次插值
- **工作方式**：计算每位同学的「赤字值」（期望次数 - 实际次数），赤字越大，概率提升越多
- **默认参数**：
  - `factor`: 13.3
  - `maxThreshold`: 3
  - `maxBoostPercent`: 1200%
- **统计展示**：数据统计页面同时显示「原始概率」和「平衡概率」两列数据

## 更新日志

设置页面底部展示应用的版本更新历史，数据从 `up.json` 文件加载。
