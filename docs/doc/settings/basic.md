# 基本设置

本节介绍 Cyreneの随机点名器的基本设置选项。

## 个性化设置

### UI 缩放

调整界面整体大小，适应不同屏幕分辨率：

- **缩放范围**：75% - 200%
- **调整方式**：拖动滑块或输入数值
- **实时预览**：调整时实时预览效果
- **保存设置**：点击 "应用" 按钮保存设置

### 字体设置

调整名字显示字体大小：

- **字体大小**：12px - 48px
- **预览效果**：在预览区查看字体大小效果
- **保存设置**：点击 "应用" 按钮保存设置

### 主题切换

切换深色/浅色模式：

- **浅色模式**：适合白天或明亮环境使用
- **深色模式**：适合夜晚或暗光环境使用
- **跟随系统**：自动跟随系统主题设置
- **切换方式**：点击主题切换按钮

### 语言切换

切换界面语言：

- **简体中文**：中文界面
- **English**：英文界面
- **切换步骤**：
  1. 在设置界面点击 "语言" 下拉菜单
  2. 选择需要的语言
  3. 点击 "应用" 按钮
  4. 界面语言立即切换，无需重启

## 功能设置

### 平衡算法

控制随机抽取的公平性：

- **启用平衡算法**：被抽中次数越少，下次被抽中的概率越高
- **禁用平衡算法**：完全随机，每次抽取概率相同
- **算法说明**：平衡算法确保长期来看每个学生被抽中的次数基本相同
- **适用场景**：教学场景建议启用，游戏场景可禁用

### English Mode

切换英文显示模式：

- **启用 English Mode**：所有界面元素显示为英文
- **禁用 English Mode**：显示中文界面
- **影响范围**：包括菜单、按钮、提示信息等
- **独立设置**：与语言设置独立，可同时使用中文界面和英文名单显示

### 动画效果

控制抽取动画：

- **彩虹名称动画**：抽取结果以彩虹渐变色显示
- **翻牌动画**：翻牌点名的 3D 翻转效果
- **禁用动画**：禁用所有动画效果，提高性能
- **动画速度**：调整动画播放速度（慢/中/快）

### 音效设置

控制抽取音效：

- **启用音效**：抽取时播放音效
- **禁用音效**：静音模式
- **音量控制**：调整音效音量
- **音效选择**：选择不同的音效文件

## 数据设置

### 存储位置

设置数据存储位置：

- **默认位置**：应用程序目录下的 data 文件夹
- **自定义位置**：选择其他文件夹作为存储位置
- **迁移数据**：将现有数据迁移到新位置
- **备份位置**：设置自动备份的存储位置

### 自动备份

配置自动备份功能：

- **启用自动备份**：定期自动备份数据
- **备份频率**：每天/每周/每月
- **备份保留**：保留最近多少个备份
- **备份内容**：选择备份哪些数据（名单、记录、设置）

### 数据清理

清理不需要的数据：

- **清理缓存**：清理临时文件和缓存
- **清理记录**：删除旧的抽取记录
- **清理名单**：删除未使用的名单
- **安全删除**：确认后永久删除数据

## 高级设置

### 性能优化

调整性能相关设置：

- **硬件加速**：启用/禁用 GPU 加速
- **内存限制**：设置应用程序内存使用限制
- **后台运行**：最小化时继续运行
- **启动选项**：设置开机自启动

### 快捷键配置

自定义快捷键：

- **默认快捷键**：恢复默认快捷键设置
- **自定义快捷键**：为常用功能设置快捷键
- **快捷键冲突**：检测并解决快捷键冲突
- **导出配置**：导出快捷键配置文件

### 更新设置

配置应用程序更新：

- **自动检查更新**：启动时自动检查新版本
- **更新渠道**：稳定版/测试版
- **下载更新**：自动下载更新包
- **安装更新**：提示安装更新

## 注意事项

- 部分设置需要重启应用程序才能生效
- 设置会自动保存在本地存储中
- 建议定期备份设置和数据
- 修改设置前建议了解每个选项的作用
