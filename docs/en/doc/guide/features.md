# Features Guide

Cyrene's Name Roller is a feature-rich random name picking desktop application that supports multiple picking modes and data management features. This document will详细介绍 all features.

## Random Name Picking Feature

### Basic Random Picking

- **Single Mode**: Pick one student at a time
- **Multiple Mode**: Pick multiple students at a time
- **No Repeat**: Pick without replacement, ensuring each person is picked only once per list
- **Balanced Algorithm**: Fewer picks result in higher probability, ensuring fairness

### Advanced Features

- **Rainbow Name Animation**: Picking results display with rainbow gradient colors, creating vivid visual effects
- **English Mode**: Support English interface display
- **List Switching**: Support multiple lists with real-time switching
- **History Records**: Save picking history for easy review

### Operation Flow

1. Select a list (optional)
2. Choose mode (Single/Multiple)
3. Click the "Start" button
4. View picking results
5. Repeat operations or switch lists

## Card Flip Name Picking Feature

### 3D Card Flip

- **Card Animation**: 3D flip effect with excellent visual experience
- **One-click Multiple**: Automatic shuffle + flip, easy to operate
- **Custom Count**: Set the number of people to pick at once
- **Card History**: Save each picking card record

### Operation Flow

1. Set the number of people to pick
2. Click the "One-click Multiple" button
3. Wait for shuffle animation to complete
4. Click cards one by one to view results
5. View card history

### Card Management

- **Real-time Switching**: Support switching lists during picking
- **History Persistence**: Card history saved in local storage
- **Batch Operations**: Support batch card collection and reset

## Data Management Feature

### Data Statistics

- **Pick Count**: Count picks for each list
- **Probability Distribution**: Display pick probability for each student
- **Balanced Probability**: Display actual probability considering balanced algorithm
- **Chart Display**: Visual display of statistical data

### Pick Records

- **Detailed Records**: Name, list, source, time for each pick
- **Filter Function**: Filter records by list, time, name
- **Search Function**: Quickly find specific records
- **Export Function**: Export records to CSV or JSON format

### List Management

#### Basic Operations
- **Add List**: Create new lists
- **Edit List**: Modify list name and description
- **Delete List**: Delete unnecessary lists
- **Batch Delete**: Support multi-select batch delete (10-second undo protection)

#### Student Management
- **Add Student**: Add student names to lists
- **Edit Student**: Modify student names
- **Delete Student**: Remove students from lists
- **Batch Import**: Batch import student names from text files
- **Export List**: Export lists to text files

#### Data Security
- **SHA-256 Encryption**: Use SHA-256 algorithm to protect sensitive data
- **Local Storage**: All data saved locally for privacy protection
- **Automatic Backup**: Support automatic backup function

## Personalization Settings

### Interface Customization

- **UI Scaling**: Adjust interface size (75%-200%) to adapt to different screens
- **Name Font Size**: Adjust the display size of names in picking results
- **Theme Switching**: Dark mode / Light mode to adapt to different environments
- **Language Switching**: Chinese / English, supporting multiple languages

### Feature Configuration

- **Balanced Algorithm Toggle**: Enable/disable balanced algorithm
- **English Mode**: Switch English display mode
- **Animation Effects**: Enable/disable rainbow name animation
- **Sound Settings**: Enable/disable picking sound effects

### Keyboard Shortcuts

- **Spacebar**: Quick start picking
- **Escape**: Cancel current operation
- **Ctrl+S**: Save current list
- **Ctrl+Z**: Undo last operation
