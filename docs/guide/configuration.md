# 配置选项

CyreneNameRoller 提供丰富的配置选项，满足不同场景的需求。

## 界面设置

### UI 缩放
- **范围**：75% - 200%
- **默认值**：100%
- **说明**：调整整体界面大小，适应不同分辨率的显示器
- **推荐**：
  - 高分辨率显示器（4K）：125% - 150%
  - 标准显示器（1080p）：100%
  - 小屏幕：90% - 100%

### 字体大小
- **范围**：1x - 3x
- **默认值**：1x
- **说明**：调整点名结果显示的字体大小
- **推荐**：
  - 投影演示：2x - 3x
  - 个人使用：1x - 1.5x

### 深色模式
- **选项**：开启 / 关闭
- **默认值**：关闭
- **说明**：切换深色/浅色主题
- **提示**：可以跟随系统设置自动切换

## 语言设置

### 界面语言
- **选项**：中文 / English
- **默认值**：中文
- **说明**：切换界面显示语言
- **注意**：切换语言后，名单中的姓名显示语言不会改变

### English Mode
- **选项**：中文 / English
- **默认值**：中文
- **说明**：点名时显示中文还是英文名字
- **位置**：随机点名页面的右上角

## 点名设置

### 随机点名

#### 抽取人数
- **范围**：1 - 100
- **默认值**：1
- **说明**：一次抽取的人数

#### 禁止重复
- **选项**：开启 / 关闭
- **默认值**：开启
- **说明**：
  - 开启：抽取后不放回，必须重置后才能再次抽取
  - 关闭：可以重复抽取同一个人

#### 彩虹动画
- **选项**：开启 / 关闭
- **默认值**：开启
- **说明**：点名结果是否显示彩虹渐变动画
- **性能提示**：关闭可略微提升性能

### 翻牌点名

#### 自动翻牌
- **选项**：开启 / 关闭
- **默认值**：关闭
- **说明**：发牌后自动翻牌，还是手动点击翻牌

#### 发牌动画速度
- **范围**：快 / 中 / 慢
- **默认值**：中
- **说明**：调整卡片发牌动画的速度

## 平衡算法设置

详细说明请参考 [平衡算法](/guide/balance-algorithm.md)。

### 启用平衡算法
- **选项**：开启 / 关闭
- **默认值**：开启
- **说明**：是否启用智能平衡功能

### 平衡因子（factor）
- **范围**：1 - 100
- **默认值**：13.3
- **说明**：定义理论阈值的分母

### 最大阈值（maxThreshold）
- **范围**：0 - 100
- **默认值**：3
- **说明**：超过此值后概率不再增加

### 最大提升百分比（maxBoostPercent）
- **范围**：200 - 5000
- **默认值**：1200
- **说明**：限制概率提升上限

### 平衡曲线点（points）
- **数量**：3 个可调点
- **格式**：`{x: number, y: number}`
- **说明**：控制 Hermite 插值曲线的形状

## 数据安全设置

### 密码保护
- **功能**：为敏感操作设置密码
- **保护操作**：
  - 清除数据
  - 删除名单
  - 批量删除人员
- **加密方式**：SHA-256 哈希
- **说明**：密码不会明文存储

### 自动备份
- **功能**：定期自动导出数据
- **频率**：每天一次
- **位置**：应用数据目录的 `backups/` 文件夹
- **说明**：可自定义备份保留天数

## 性能设置

### 粒子效果
- **选项**：开启 / 关闭
- **默认值**：开启
- **说明**：点名时是否显示粒子特效
- **性能提示**：关闭可显著提升性能

### 模糊效果
- **选项**：开启 / 关闭
- **默认值**：开启
- **说明**：是否启用亚克力材质模糊效果
- **性能提示**：关闭可提升性能，但降低视觉效果

### 动画速度
- **范围**：50% - 200%
- **默认值**：100%
- **说明**：全局调整所有动画的速度

## 高级设置

### 调试模式
- **选项**：开启 / 关闭
- **默认值**：关闭
- **说明**：启用开发者工具，方便调试
- **警告**：仅开发者使用

### 日志级别
- **选项**：Error / Warn / Info / Debug
- **默认值**：Info
- **说明**：控制控制台输出的日志详细程度

### 禁用更新检查
- **选项**：开启 / 关闭
- **默认值**：关闭
- **说明**：禁用自动更新检查

## 配置文件位置

### Windows
```
%APPDATA%/CyreneNameRoller/settings.json
```

### macOS
```
~/Library/Application Support/CyreneNameRoller/settings.json
```

### Linux
```
~/.config/CyreneNameRoller/settings.json
```

### 配置文件结构
```json
{
  "appearance": {
    "darkMode": false,
    "uiScale": 100,
    "fontSize": 1
  },
  "language": {
    "lang": "zh",
    "displayLang": "zh"
  },
  "balance": {
    "enabled": true,
    "factor": 13.3,
    "maxThreshold": 3,
    "maxBoostPercent": 1200,
    "points": [
      { "x": 0.3, "y": 150 },
      { "x": 1.5, "y": 420 },
      { "x": 2.4, "y": 800 }
    ]
  },
  "performance": {
    "particles": true,
    "blur": true,
    "animationSpeed": 100
  },
  "security": {
    "passwordHash": "",
    "autoBackup": true,
    "backupDays": 7
  }
}
```

## 重置配置

如果配置出现问题，可以在**设置**页面点击**恢复默认设置**按钮：

1. 进入 **设置** 页面
2. 滚动到底部
3. 点击 **恢复默认设置**
4. 输入密码（如果设置了）
5. 确认操作

**注意**：此操作不可撤销！

## 导入/导出配置

支持导出和导入配置文件，方便在不同设备间同步设置：

### 导出配置
1. 进入 **设置** 页面
2. 点击 **导出配置**
3. 选择保存位置

### 导入配置
1. 进入 **设置** 页面
2. 点击 **导入配置**
3. 选择配置文件
4. 确认导入

## 常见配置场景

### 课堂点名（推荐）
```json
{
  "balance": {
    "enabled": true,
    "factor": 13.3,
    "maxThreshold": 3,
    "maxBoostPercent": 1200
  },
  "appearance": {
    "fontSize": 2,
    "uiScale": 100
  }
}
```

### 团队游戏（激进的平衡）
```json
{
  "balance": {
    "enabled": true,
    "factor": 8,
    "maxThreshold": 5,
    "maxBoostPercent": 2000,
    "points": [
      { "x": 0.2, "y": 250 },
      { "x": 0.8, "y": 600 },
      { "x": 1.5, "y": 1200 }
    ]
  }
}
```

### 公平抽奖（无平衡）
```json
{
  "balance": {
    "enabled": false
  },
  "appearance": {
    "fontSize": 3
  }
}
```

### 低性能设备
```json
{
  "performance": {
    "particles": false,
    "blur": false,
    "animationSpeed": 150
  }
}
```
