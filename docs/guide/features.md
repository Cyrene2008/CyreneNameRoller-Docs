# 功能详解

## 随机点名

### 基础操作
1. **选择名单**：从下拉菜单选择要抽取的名单
2. **设置人数**：调整一次抽取的人数（1-100）
3. **开始抽取**：点击"开始"按钮开始滚动
4. **查看结果**：等待动画结束后显示结果

### 高级选项

#### English Mode
切换中英文名字显示：
- 切换开关后，点名结果会显示对应语言的名字
- 如果某个人只有中文名，切到英文模式会显示中文名
- 反之亦然

#### 多人模式
一次抽取多人的两种方式：
- 直接设置抽取人数
- 每次点击"再来一次"不重置，累计抽取

#### 禁止重复
- 开启：抽取后不放回，需要重置后才能再次抽取
- 关闭：可以重复抽取同一个人

### 动画效果
点名结果展示时会有多种动画效果：

1. **滚动阶段**：名字快速滚动
2. **减速阶段**：滚动速度逐渐减慢
3. **最终结果**：显示最终抽取的名字
4. **彩虹动画**：名字显示彩虹渐变效果（可关闭）

### 重置选项
- **重置名单**：清空当前抽取记录，恢复初始状态
- **重置所有**：清空所有抽取记录，包括统计

## 翻牌点名

### 发牌流程
1. **选择名单**：选择要抽取的名单
2. **设置人数**：设置发牌数量
3. **发牌**：点击"发牌"按钮
4. **翻牌**：点击卡片进行翻转

### 发牌动画
卡片从随机方向飞入，带有旋转和缩放效果：
- 发牌速度可在设置中调整
- 可选择自动翻牌或手动翻牌

### 收牌历史
- 翻牌后自动记录到历史
- 切换名单时历史不会丢失
- 可以手动清空历史

### 批量操作
- **一键多抽**：设置人数后一次性发多张牌
- **自动洗牌**：每次发牌前自动洗牌
- **收牌重置**：一键收牌并重置状态

## 名单管理

### 创建名单
1. 进入 **名单管理** 页面
2. 点击 **新建名单**
3. 输入名单名称
4. 添加人员

### 添加人员
支持三种方式：

#### 手动输入
逐个添加人员：
1. 输入中文名
2. 输入英文名（可选）
3. 点击添加按钮

#### 批量导入
从文本文件导入：
1. 准备文本文件，每行一个名字
2. 点击 **导入**
3. 选择文件
4. 确认导入

#### 从其他名单复制
从已有名单复制人员：
1. 选择源名单
2. 选择要复制的人员
3. 点击复制
4. 粘贴到目标名单

### 编辑人员
- 点击人员行进入编辑模式
- 修改中文名或英文名
- 保存更改

### 删除人员
支持两种删除方式：

#### 单个删除
1. 点击人员行的删除按钮
2. 确认删除
3. 删除成功

#### 批量删除
1. 选择要删除的人员
2. 点击 **批量删除**
3. 确认删除
4. **10 秒内可撤销**

### 白名单
白名单是特殊人员列表，他们**不会被平衡算法影响**：
- 白名单人员的概率始终相等
- 即使被抽中多次，也不会增加概率
- 适合需要保证公平性的特殊角色

### 导入/导出名单
支持 `.cyrene` 格式的名单文件：

#### 导出
1. 选择要导出的名单
2. 点击 **导出**
3. 选择保存位置
4. 导出成功

#### 导入
1. 点击 **导入**
2. 选择 `.cyrene` 文件
3. 确认导入
4. 导入成功

## 数据统计

### 统计指标

#### 抽取次数
每个人被抽中的总次数。

#### 原始概率
不考虑平衡算法的理论概率：
```
原始概率 = 1 / 总人数
```

#### 平衡概率
考虑平衡算法后的实际概率：
```
平衡概率 = 原始概率 × (100 + 提升百分比) / 100
```

### 排序和筛选
- 按抽取次数排序
- 按原始概率排序
- 按平衡概率排序
- 按姓名筛选

### 数据可视化
统计页面提供直观的可视化展示：
- 柱状图显示抽取次数分布
- 饼图显示抽取比例
- 折线图显示概率变化

## 抽取记录

### 记录内容
每次抽取都会记录：
- **姓名**：被抽中的人名
- **名单**：使用的名单名称
- **来源**：随机点名 / 翻牌点名
- **时间**：抽取的具体时间

### 查看记录
- 按**时间**筛选
- 按**名单**筛选
- 按**来源**筛选
- 搜索姓名

### 导出记录
支持导出为 CSV 或 JSON 格式：
1. 点击 **导出记录**
2. 选择格式（CSV/JSON）
3. 选择保存位置
4. 导出成功

### 清空记录
1. 点击 **清空记录**
2. 输入密码（如果设置了）
3. 确认清空
4. 记录清空

## 高级技巧

### 快速切换名单
- 在点名页面，使用下拉菜单快速切换
- 切换后自动保留抽取状态

### 批量抽取
- 设置抽取人数后连续抽取
- 适合一次性抽取多人

### 历史回溯
- 抽取记录支持按时间回溯
- 方便查看历史抽取情况

### 数据备份
- 定期导出数据备份
- 避免意外数据丢失

### 性能优化
- 关闭彩虹动画和粒子效果可提升性能
- 在低性能设备上建议禁用模糊效果
