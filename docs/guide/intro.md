# 项目介绍

**CyreneNameRoller** 是一个功能完善的随机点名桌面应用，采用 Vue 3 + Electron 构建，使用 Windows 11 Fluent Design 设计语言，提供流畅的用户体验。

## 核心特性

### 🎲 随机点名
- **多人模式**：支持一次抽取多人，适合课堂活动、团队游戏等场景
- **禁止重复**：采用不放回抽取方式，确保每位参与者都有公平机会
- **平衡算法**：内置智能加权算法，被抽中次数越少的人，下次被抽中的概率越高
- **彩虹名称动画**：抽取结果展示时带有炫酷的彩虹渐变效果
- **English Mode**：支持中英文切换，国际化友好

### 🃏 翻牌点名
- **3D 翻转动画**：卡片翻转时呈现流畅的 3D 视觉效果
- **一键多抽**：支持批量抽取，自动洗牌+翻牌
- **收牌历史持久化**：翻牌记录自动保存，切换名单不丢失
- **名单实时切换**：随时切换不同名单，保持翻牌状态

### 📊 数据管理
- **数据统计**：实时显示抽取次数、原始概率、平衡概率
- **抽取记录**：完整记录每次抽取的姓名、名单、来源和时间
- **名单管理**：支持增删改查、批量删除（10 秒撤销）、导入导出
- **数据安全**：SHA-256 密码保护关键操作，数据导出/导入/清除功能

### 🎨 个性化设置
- **UI 缩放**：支持 75%-200% 缩放比例，适应不同显示器
- **字体大小**：可调整点名结果字体大小
- **主题切换**：深色/浅色模式自由切换
- **多语言**：中/英文界面切换

## 技术栈

- **核心框架**：Vue 3 + Vite + Electron
- **状态管理**：Pinia
- **路由**：Vue Router（Hash 模式）
- **图标库**：@iconify/vue (Fluent UI Icons)
- **样式**：CSS 变量 + 自定义组件库
- **数据持久化**：Electron 文件系统 + localStorage 双写

## 项目亮点

### 智能平衡算法
CyreneNameRoller 采用基于 Fritsch-Carlson 单调三次 Hermite 插值算法的平衡系统，确保点名公平性：

- 被抽中次数少的参与者会获得更高的抽取概率
- 支持自定义平衡曲线，满足不同场景需求
- 可视化曲线编辑器，直观调整算法参数

### 数据双保险
采用 Electron 文件系统和 localStorage 双写机制，确保数据安全：

- 应用意外崩溃时自动恢复数据
- 支持数据导出备份
- 密码保护敏感操作

### 现代化设计
遵循 Windows 11 Fluent Design 设计语言：

- 亚克力材质效果（Acrylic Material）
- 光照效果（Reveal Highlight）
- 流畅的动画过渡
- 响应式布局
